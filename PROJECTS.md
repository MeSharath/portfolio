### BookMyStudio (Nacho)
**Dance Studio Booking Platform - Marketplace MVP**

**Overview:**
Building a digital marketplace platform that connects dance studio owners with students looking for classes. The platform aims to streamline the booking process and help studios manage schedules efficiently.

**Tech Stack:**
- **Frontend:** React with TypeScript
- **Backend:** Node.js with Express.js
- **Database:** PostgreSQL
- **Cloud:** Google Cloud Platform (GCP)
- **DevOps:** Docker, GitHub Actions for CI/CD
- **Tools:** n8n for workflow automation

**Key Features:**
- Studio profile management
- Class scheduling and management
- Real-time booking system
- Payment integration
- User reviews and ratings
- Automated notifications
- Analytics dashboard

**Learnings & Challenges:**
- Marketplace design patterns
- Handling concurrent bookings
- Payment processing integration
- Product-market fit research
- Cloud cost optimization

**Status:** In active development - MVP phase
